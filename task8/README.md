# AI Movie Recommendation System

## Overview
This project recommends movies based on the selected movie using Content-Based Filtering and Cosine Similarity.

## Features
- Movie recommendation
- Interactive Streamlit interface
- Fast recommendation
- Simple and user-friendly

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Dataset
MovieLens Small Dataset

## Run the Project

python train_model.py

streamlit run app.py
